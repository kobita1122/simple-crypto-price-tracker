# Simple Crypto Price Tracker

This is a beginner-friendly crypto project.

This project helps you:
- Check live crypto prices
- View price in USD
- Update prices with one click
- Understand how crypto APIs work

This project uses:
- HTML
- JavaScript
- Public crypto API

## Features
- Show Bitcoin price
- Show Ethereum price
- Refresh button
- Clean and simple code
- Easy to edit

## Requirements
- Internet connection
- Basic knowledge of HTML and JavaScript

## How to Run
1. Download this repository
2. Open `index.html` in your browser
3. Click "Refresh Prices"
4. See live crypto prices on screen

## File Structure
- index.html -> Main webpage
- app.js -> Main logic
- api.js -> API calls
- config.js -> App settings
- btc.js -> Bitcoin logic
- eth.js -> Ethereum logic
- ui.js -> Screen updates
- utils.js -> Helper functions
- format.js -> Price formatting
- refresh.js -> Button logic

## Learning Goal
This project is made to help beginners understand:
- How crypto price APIs work
- How to read JSON data
- How to update UI using JavaScript
